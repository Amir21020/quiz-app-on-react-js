# Quiz App

Простое приложение-викторина на React.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Линтинг

```bash
npm run lint
```

## Структура

- `src/assets/data.js` — вопросы и ответы
- `src/components/Quiz/` — компонент викторины
- `index.html` — точка входа
